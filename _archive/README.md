# Archive

These files are kept for reference because they came from the earlier
storage-water quality paper direction.

They are not the current workflow. Use the main repo scripts first:

- `run_active_workflow.R`
- `03_stats/`
- `04_PCA/`
- `05_synthesis/`
- `06_figures/`

## Folders

- `deprecated_storage_proxy_workflow/`: older work that used water-quality
  storage proxies before the finalized storage-paper metrics were available.
- `noncore_exploratory_workflow/`: older exploratory plots, cluster checks,
  season-timing checks, extra synchrony comparisons, and mixed-effects
  summaries.

Use archive scripts only when checking old ideas or recovering code. Do not use
them as the main analysis unless the old approach is being revisited on purpose.
